# blog.drewclark.io

This project is a personal blog built with a React front-end and a Node.js backend, styled using Tailwind CSS.

## Project Structure

- **client**: Contains the React application.
- **server**: Contains the Node.js server.

## Technologies Used

- React
- Node.js
- Tailwind CSS

## Setup Instructions

1. Clone the repository.
2. Navigate to the `client` and `server` directories and install dependencies using `npm install`.
3. Start the client and server with `npm start`.

## Features

- Responsive design
- RESTful API
